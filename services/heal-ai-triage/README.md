# HEAL AI Triage

Deterministic pre-IA triage for `gene_module_v2` match outputs.

Input:
- `sheet_final_consolidated.csv` generated by the gene-module VCF-canon matcher.

Outputs:
- `heal_fon_ai_triage.csv`
- `heal_fon_ai_triage_excluded_audit.csv`
- `ai_triage_summary.json`

This step is intended to reduce the v2 match universe before any expensive downstream interpretation work.
